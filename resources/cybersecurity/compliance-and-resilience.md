# Cybersecurity Compliance and Resilience

## Security Guidance

- [CNIL Practice Guide for the Security of Personal Data: 2024 edition](https://www.cnil.fr/en/practice-guide-security-personal-data-2024-edition)
  Brief description: A practical CNIL guide on security measures for personal data processing, updated in 2024 with new factsheets on artificial intelligence, cloud computing, mobile applications, APIs, and data management security.
  Topic: Cybersecurity, GDPR Article 32, compliance audits

- [NIST SP 800-53 Rev. 5, Security and Privacy Controls for Information Systems and Organizations](https://csrc.nist.gov/pubs/sp/800/53/r5/upd1/final)
  Brief description: NIST's control catalog for security and privacy programs, widely used to structure risk management, internal controls, assessments, and audit-ready governance.
  Topic: Security controls, privacy controls, audits, governance
